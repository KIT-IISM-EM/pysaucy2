About¶

Introduction¶

Pysaucy2 is a Python wrapper for the very efficient saucy algorithm for computing the automorphism group of a graph. Please consider saucy’s website for information on the algorithm itself and for requesting the source code (which is not included in pysaucy2) from the authors.
